# 프로가 되기 위한 웹 기술 입문
07월 17일 : 0장 ~ 2장

## 서블릿
자바로 만들어진 웹 콘텐츠를 생성하기 위한 프로그램

## 라이브러리 / 프레임워크
### 라이브러리
프로그램에서 사용할 수 있는 미리 컴파일된 루틴, 클래스 또는 인터페이스 모음이다. 일반적인 작업을 더 쉽게 적용할 수 있도록 클래스 또는 메서드 집합을 제공한다.

- jQuery

### 프레임워크
특정 플랫폼에 맞는 프로그램을 구축할 수 있는 기반을 제공하는 포괄적인 개념이다.
- Spring, Django, NestJS

### 차이점
핵심은 "제어의 역전"이다.
- 라이브러리는 사용자가 제어권을 갖고 사용시기를 결정한다.
- 프레임워크는 프레임워크가 제어권을 갖고 코드를 언제 사용할지 결정한다.

## 웹 프레임워크의 필요성
과거 인터넷이 처음 등장했을 때는 단순하고 정적인 페이지를 주로 처리했다. 하지만 현대에 이르러 웹 애플리케이션은 복잡하고 동적인 처리를 하는 페이지가 필요해졌다. 그동안의 기술로는 단순 반복작업을 처리하기 힘들었고 작업을 자동화하여 구조적이고 효율적인 처리를 위해 프레임워크가 도입되었다.

프레임워크의 도입은 애플리케이션의 구조를 빠르게 이해할 수 있도록 하였고 협업과 유지관리가 더욱 용이해졌다.
